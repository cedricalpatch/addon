Grumman F-14D Supertomcat
-------------------------


Includes 3 detailed textures of the movie "Top Gun". The skins in the movie are based on real squadrons, but have a fictional appearance.

Skin 1: "Iceman" and "Slider"

Skin 2: "Maverick" and "Goose"

Skin 3: "Cougar" and "Merlin"

Skin 4: Some special skin, because I didn't know any other TG skin I could add.



Mod features:
Detailed model
Detailed cockpit
Working gear
Working variable wings
Working elevators
Enterable for 4 persons
Bullet impact
Custom spec map
Working lights
Custom LOD models
Custom Collision model

Install

You have to use Spark IV to install this mod (google is your friend). Open Your GTA/pc/models/cdimages/vehicles.img and import the .wft and .wtd files from this archive. Rebuild the .img and save.

You have to use a custom .ini file for the script, download the script and use the .ini file inside:
Scripts/JetFiles/Custom jets/F14D



To use all skins, you have to enable multi livery in your simple native trainer (google or youtube offers tutorials)
